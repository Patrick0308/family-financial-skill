# 家庭财务记录 Skill 设计文档

**日期**：2026-06-20
**Skill 名称**：`family-financial`
**类型**：Claude Code 个人 skill（安装于 `~/.claude/skills/family-financial/`）

## 1. 目标

帮助家庭用自然语言记录财务状况，自动生成三张规范报表，并给出健康评分与财富等级：

- **家庭资产负债表** —— 某一时点的资产、负债、净资产快照
- **家庭收入支出表** —— 某周期内的收入、支出、结余
- **家庭现金流表** —— 某周期内的经营/投资/筹资现金流与净现金流
- **财务健康评分** —— 0–100 分 + 等级 + 通用理财建议（基于财务比率）
- **家庭财富等级** —— 基于可投资净资产的分档（综合总净资产、年收入参考）

## 2. 核心设计决策（已确认）

| 决策点 | 选择 |
|---|---|
| 数据存储 | 本地文件（CSV + Markdown），存于项目目录 |
| 记录方式 | 混合：日常收支记流水自动汇总；资产负债定期手动更新快照 |
| 记录周期 | 按月为基础粒度，可聚合到季/年 |
| 分类体系 | 预设标准家庭分类打底，可自定义 |
| 收支 vs 现金流 | 每笔流水标 `类型`（收入/支出/转移），收支表只用收入支出，现金流表用全部三类 |
| 财富等级口径 | 以「可投资净资产」为主轴，总净资产与年收入作同屏参考 |
| 币种 | 人民币（¥），单币种 |

## 3. 数据文件结构

数据存放在运行 skill 时所在项目目录下的 `data/`，生成的报表存于 `reports/`：

```
family-financial/
├── data/
│   ├── transactions.csv     # 收支/转移流水（追加写入）
│   ├── balances.csv         # 资产负债时点快照
│   └── categories.md        # 可自定义的分类表（预设打底，含资产标签）
└── reports/
    └── 2026-06.md           # 生成的月度报表（三表 + 评分 + 等级）
```

### 3.1 transactions.csv

每笔流水一行，列：

| 列名 | 说明 | 取值/示例 |
|---|---|---|
| 日期 | 交易日期 | `2026-06-20` |
| 类型 | 收入 / 支出 / 转移 | `支出` |
| 现金流分类 | 经营 / 投资 / 筹资 | `经营` |
| 方向 | 流入 / 流出 | `流出` |
| 分类 | 来自 categories.md | `餐饮` |
| 金额 | 正数（¥） | `120` |
| 账户 | 可选 | `招行信用卡` |
| 备注 | 可选 | `买菜` |

**类型与现金流分类的默认映射**（skill 自动判断，可被用户覆盖）：

- `收入` → 经营 / 流入（如工资、奖金）；投资收益归 `投资`
- `支出` → 经营 / 流出（日常消费）
- `转移` → 投资（买卖股票基金、买房自用除外）或筹资（借入、还本金）

### 3.2 balances.csv

资产负债快照，每项一行，列：

| 列名 | 说明 | 取值/示例 |
|---|---|---|
| 日期 | 快照时点 | `2026-06-30` |
| 类型 | 资产 / 负债 | `负债` |
| 项目 | 来自 categories.md | `房贷` |
| 金额 | 正数（¥） | `800000` |

出表时取目标月份**最近一次**快照。

### 3.3 categories.md（含资产标签）

预设分类打底，用户可增删。资产项目额外带两个标签：

- **流动性**：`流动` / `非流动` —— 用于应急储备计算
- **性质**：`可投资` / `自用` —— 用于可投资净资产计算

预设分类：

- **资产**
  - 现金/活期 —— 流动、可投资
  - 定期存款 —— 流动、可投资
  - 投资（股票/基金） —— 流动、可投资
  - 房产（自住） —— 非流动、自用
  - 房产（投资） —— 非流动、可投资
  - 车辆（自用） —— 非流动、自用
  - 其他资产 —— 非流动、自用
- **负债**：房贷、车贷、信用卡、消费贷、其他
- **收入**：工资、奖金、投资收益、其他
- **支出**：餐饮、居住、交通、教育、医疗、娱乐、人情、其他

## 4. Skill 操作（自然语言触发）

| 用户说 | 行为 |
|---|---|
| "记一笔：今天买菜 120" | 追加流水，自动判断类型/分类/现金流分类/方向 |
| "工资到账 20000" | 追加流水（收入/经营/流入/工资） |
| "更新余额：房贷剩 80 万、活期 5 万…" | 写一组快照到 balances.csv |
| "出 6 月的表" | 生成 reports/2026-06.md（三表 + 评分 + 等级） |
| "看二季度汇总" | 聚合月度数据生成季度视图 |
| "加个分类：宠物" | 在 categories.md 增项 |

记账时若类型/分类判断不确定，skill 应向用户确认而非臆测。

## 5. 三张报表

### 5.1 资产负债表（时点）

- 列出资产各项与合计
- 列出负债各项与合计
- **净资产 = 总资产 − 总负债**

### 5.2 收入支出表（周期）

- 收入按分类汇总 → 收入合计
- 支出按分类汇总 → 支出合计
- **月结余 = 收入合计 − 支出合计**
- 数据来源：仅 `收入` 和 `支出` 类流水

### 5.3 现金流表（周期）

- **经营性净现金流**：经营类流入 − 流出
- **投资性净现金流**：投资类流入 − 流出
- **筹资性净现金流**：筹资类流入 − 流出
- **净现金流合计 = 三者之和**
- 数据来源：全部三类流水

## 6. 财务健康评分

基于报表计算公认家庭财务比率，每项打分，加权汇总为 **0–100 总分 + 等级 + 通用建议**。
评分公式、阈值、权重写在 `references/scoring.md`，可调。

| 指标 | 公式 | 健康参考 |
|---|---|---|
| 结余比率 | 月结余 / 月收入 | ≥ 30% |
| 资产负债率 | 总负债 / 总资产 | ≤ 50% |
| 偿债收入比 | 月还债额 / 月收入 | ≤ 40% |
| 应急储备 | 流动资产 / 月支出 | 3–6 个月 |
| 投资资产比 | 可投资资产 / 总资产 | 适度 |

**等级映射**：优秀（≥85）/ 良好（70–84）/ 一般（55–69）/ 需改善（<55）。

**重要约束**：建议文案限于**通用家庭理财常识**（如"应急储备建议覆盖 3–6 个月支出"），
**不做个性化投资推荐**（不推荐具体标的、不做买卖择时建议）。skill 明确声明自己非持牌顾问。

## 7. 家庭财富等级

以**可投资净资产**为主轴分档，同屏显示总净资产与年收入作参考，并给出"距下一档差额"。

**默认口径**（透明、可在 `references/wealth-tiers.md` 修改）：

```
可投资净资产 = 可投资资产 − 非自住房负债（信用卡/消费贷/车贷等）
（即把自住房产与对应房贷整体剔除）
总净资产     = 总资产 − 总负债
```

**默认分档**（人民币，经验值，非官方标准，可自定义）：

| 等级 | 可投资净资产参考区间 |
|---|---|
| 起步 | < 10 万 |
| 稳健 | 10 万 – 100 万 |
| 小康 | 100 万 – 500 万 |
| 富裕 | 500 万 – 1000 万 |
| 高净值 | 1000 万 – 1 亿 |
| 超高净值 | > 1 亿 |

阈值来源说明（写入 references）：参考高净值/私行「可投资资产」口径与国内城镇家庭资产负债调查，
属经验分档，非权威发布，鼓励用户按自身地区调整。

报表结尾两行结论示例：

> **财务健康评分：78 / 100（良好）**
> **家庭财富等级：小康（可投资净资产 ¥1,250,000；总净资产 ¥2,050,000；年收入 ¥240,000；距「富裕」还差 ¥3,750,000）**
